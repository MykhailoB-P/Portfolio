# IT Management & Cybersecurity Portfolio Website

Welcome to the source code for my personal professional portfolio website. I built this static website to showcase my practical skills, academic projects, and technical versatility in IT infrastructure, scripting, and cybersecurity.

## Portfolio Features
This website is built as a fast, secure, and responsive static web application using clean **HTML5, CSS3, and JavaScript**. It does not require a database, making it completely secure from SQL injection attacks and lightweight to host.

Key features include:
* **Interactive Project Grid:** A custom JavaScript filter bar that allows visitors to sort projects by category (*Cybersecurity*, *Software Engineering*, and *Infrastructure*) without reloading the page.
* **Smooth UI/UX:** Dark-themed professional design with high contrast, colored category badges, and smooth animations powered by AOS (Animate On Scroll).
* **Direct Proof of Work:** Expandable project cards with direct links to downloadable PDF technical reports and GitHub repositories.
