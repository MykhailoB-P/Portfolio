My Journey into IT

My interest in technology started with a simple question: How these technologies even work? That curiosity evolved into building local virtualized labs, configuring Linux servers, and exploring network packet flows.

Today, I specialize in combining IT infrastructure management with proactive cybersecurity practices. Whether it is deploying secure web environments, automating routine administrative tasks with Python and PowerShell, or analyzing network security logs, I focus on practical execution over theory.

As I prepare for industry roles in IT Support, SysAdmin, and Security Operations, this portfolio serves as a live demonstration of my projects, lab setups, and technical growth.
